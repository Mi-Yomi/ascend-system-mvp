# Ascend System MVP

Solo Leveling-inspired productivity system MVP for Anuar.

## What it has
- Quest dashboard
- XP / Level / Rank progression
- Stats system
- Penalty quests on failure
- System log feed
- Local demo persistence via localStorage

## Stack
- Static HTML/CSS/JS MVP
- Easy to wrap in Flutter WebView later
- Supabase-ready architecture can be added next
